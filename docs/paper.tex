\section{Introduction}
Reflectiveness is a powerful concept that allows interpreters to be manipulated by
parent interpreters. This paper presents how this idea can be extended to type systems,
by treating more strict type systems as childs of more ``loose'' type systems. This
allows a parent type system to change the semantics of its child type system.
